## 运行
* 本项目使用了customtkinter以及requests库，所以应该首先安装这两个库
    ```shell
    pip install customtkinter
    pip install requests
    ```
* 然后运行main.py即可
    ```shell
    python main.py
    ```
  
## 项目结构
* main.py: 主程序
* GUI.py: GUI界面
